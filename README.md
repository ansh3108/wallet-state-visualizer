# Wallet State Visualizer

A dev tool tha visualizes what a Solana wallet actually is at the account level.

This is **not** a portfolio app and not a balance checker.


# What It Shows

* Wallet Connection (Phantom)
* System account details:
  * SOL Balance
  * Lamports
  * Owner program
  * Data size
  * Rent-exempt status

# Tech Stack

* Wallet Connection
* web3.js
* Phantom Wallet
* Devnet RPC
